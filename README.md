# Linux-Mini-Games
  This repo has some mini games that you can play on linux

# Pre-requsite:
  You must have gnu g++ installed.

# How to play:
1. download source code of the game.
2. open terminal.
3. type cd Downloads.
4. type g++ {file_name} eg - g++ nim.cpp
5. type ./a.out
6. go play.
